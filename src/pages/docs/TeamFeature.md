## Team3

Gunakan perintah `/team3` untuk bergabung dengan team 3.

### Contoh Penggunaan
```bash
/team3
```

### Response Success
```
Bot Akan Menginvite Di Ingame
Terima Invite untuk bergabung
kedalam Team 3
```
